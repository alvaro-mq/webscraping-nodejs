# webscraping-nodejs
Webscraping para pagina https://boliviaverifica.bo

## Tecnologias
- Nodejs
- Puppeteer
- Cheerio

## Instrucciones
1. Instalar dependencias
```
$ npm install
```
2. Configurar parametros en index.js
```
- category
- initPage
- endPage
```
3. Iniciar el proceso
```
$ node index.js
```
